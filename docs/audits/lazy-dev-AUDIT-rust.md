> **Prominent guardrail:** the task's "Python live path replaces the Rust compressors" hypothesis is **empirically false.** The Python files are **thin delegating shims**, not a second implementation.

- **Verified:** `_core` registers `PyDiffCompressor`, `PySearchCompressor`, `PySmartCrusher`, `PyLogCompressor` (`lib.rs:1630–1644`). The ground-truth's `#[pyfunction]`-only trace **missed** these `#[pyclass]`/`#[pymethods]` entries.
- **Do NOT cut:** `diff_compressor.rs` (1,685), `log_compressor.rs` (1,451), `search_compressor.rs` (877), `smart_crusher/` subsystem. Archive **nothing** here.
